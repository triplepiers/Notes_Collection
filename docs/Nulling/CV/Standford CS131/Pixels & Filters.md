## 1 Color

### 1.1 Color Space 色域

- Linear Color Space
	- 色彩由三原色的权重决定
		- 两种颜色混合的结果在其连线上
		- 三种颜色混合的结果在其围成的三角形内
	- 例子：RGB Space - 三种单色光的线性组合表示颜色
- Nonlinear Color Space
	- 例子：HSV（ RGB是其在100%明度下的特例），使用三个维度描述元素
		- Hue 色相
			  0-360 转一圈就是 RGB 色域（从红色开始）
		- Saturation 饱和度
			  从中心的白色过渡到最骚的颜色（光谱色）
		- Value 明度
			  为 0 时全黑
