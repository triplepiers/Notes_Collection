    1 JS 实现

    完整的 JS 实现包含以下几个部分： - 核心 - ECMAScript 描述了语法、类型、语句、关键字、保留子、操作符、全局对象 - 文档对象模型 - DOM - 浏览器对象模型 - BOM 针对浏览器窗口及其子窗口

    2 HTML 中的 JS 实现

    2.1 <script> 标签

    该标签拥有下列 8 个属性： - async - 立即下载脚本，但不阻断其他页面动作（只对外部脚本有效） - 不保证执行顺序 - crossorigin：配置 CORS，默认不使用 - crossorigin= "anonymous" 文件请求不需要设置凭据标志 - crossorigin="use-credentials 出站请求包含凭据 - defer - 立刻下载，但在整个页面解析完成后执行 - 保证按出现顺序执行：首个 defer 必然在第二个 defer 之前执行

    • FACT

      • 可以通过转义字符规避行内脚本异常结束 <\/script>
      • 使用 src 引入外部文件的 <script> 文件不应该包含行内代码 浏览器只会下载并执行 src 指向的脚本文件，并忽略行内代码
      • <script> 标签允许指向一个 跨域 的文件
      • 当不存在 defer / async 属性时，浏览器将按照书写顺序依次解释 <script> 只有上一个解释完毕后，才会顺序解释下一个

    • 标签位置

      • 将所有脚本都放在 <head> 中可能会导致页面渲染明显延迟 因为需要下载、解析并解析完成所有 JS 脚本后，再开始渲染、
      • 现在一般放在 <body> 内页面内容之后的地方

    3 语言基础

    3.1 特性

    • 区分大小写：变量、函数、操作符 均区分大小写
      • test & Test 是两个不同的变量
      • typeof 是保留词，但 Typeof 可以作为有效函数名
    • 标识符：变量 / 函数 / 属性 / 参数 名称
      • 首字符必须是 字母 / 下划线 / $
      • 其余字符可以是 字母 / 下划线 / $ / 数字
      • 字符可以是 ASCII 字母 / Unicode 字母
      • 一般使用小驼峰（第一个小写，后面首字母大写）
      • 保留字不能作为标识符

    3.2 变量

    • var - 范围是「函数作用域」
      • 定义可以保存任何类型值的变量，未初始化时为 undefined
      • 会成为 window 对象的属性，可以通过 window.name 进行访问
      • 可以在随后改变 值 / 值的类型
      • 允许通过 , 分隔定义的多个变量，并分别进行初始化
      • 作用域
        • 使用 var message 在函数内 定义的变量是局部变量，将在推出函数时被销毁
        • 省略 var 关键词将定义一个 全局变量，可以在函数外进行访问
      • 「声明提升」 会把所有变量声明 拉到函数作用域顶部 - 允许先使用后声明 可以重复声明同一个变量 => 使用截止至目前的最新值
    • let - 范围是「块作用域」=> 在最近的大括号内有效
      • 不允许同一块内重复声明
      • 不会成为 window 对象的属性 但是声明仍是在全局作用域中发生的，响应变量将在页面生命周期中存续 => 最好还是不要重复声明变量
      • 下面的声明是被允许的（有不同的作用域） 
        let age = 30;
console.log(age) // = 30
if(true) {
    let age = 26;
    console.log(age) // = 26 作用域不同不冲突
}
      • 混用 var & let 使用同一变量一样会导致冲突 只是指出了作用域范围，声明的还是同类型的变量 
        var name
let name // => 冗余声明了变量 name
      • 「暂时性死区」- 在 let 声明前的执行瞬间 let 声明的变量在作用域中不会被提升 js console.log(age) // => ReferenceError：age 未定义 let age
      • For 循环中的 let 变量 for(let i = 0; i < 5; i++) => 变量 i 仅限在循环内使用 每一次迭代将声明一个新的迭代变量 -> 每次循环都会引用不同的变量实例
    • const
      • 必须在 声明的同时初始化变量
      • 再次修改 const 变量将导致运行时错误 修改 变量内部 的属性不违反限制
      • 不允许重复声明
      • 作用域是「块」
      • 不能用于声明迭代变量，因为自增操作 = 给常量重新赋值

    3.3 数据类型

    • 六种原始类型 Undefined Null Boolean Number String Symbol
    • 复杂数据类型 Object
    • typeof 操作符将返回以下几种值 - typeof message undefined boolean string number object function symbol

    Undefined

    • 当使用 let / var 声明变量但没有进行初始化时，相当于给变量赋 undefined
    • 对 未声明 / 未初始化 的变量执行 typeof 均返回 undefined
    • 在进行条件判断时，undefined == false 我们可以用 if(name) 判断变量是否定义 / 是否赋初值

    Null

    • 是一个 空的对象指针，所以 typeof null = Object
    • null == undefined 返回 true
    • 需要进行显式赋值来保持空对象指针的语义
    • 进行条件判断时，null == false

    Boolean

    • 区分大小写，false / true 是有效值
    • 可以使用 Boolean() 将非 boolean 类型转换为 boolean
      • false 空字符串、0、NaN、null、undefined
      • true 非空字符串、非零数值（包括无穷），任意非空 Object

    Number

    let intNum = 55 // 十进制整数
let Oct_1 = 070 // 0开头的八进制数
let Oct_2 = 079 // 9无效，识别为十进制79
let Hex_1 = 0xA // 0x开头的十六进制数，前缀区分大小写
let float_1 = .1 // 小数点 + 小数点后一位 ｜ 小数点前可省略 0
// 因为浮点值的存储空间为整数的两倍，以下形式将被转为整数
// 最高精度是 17 位小数
let f_1 = 1. // 小数点后没有数字，当成整数 1
let f_2 = 10.0 // 小数点后为0，当成整数 10
// 支持科学计数法
let floatNum_1 = 3.125e7 // 等价于 3.125*10^7
let floatNum_2 = 3e-17 // 等价于 3*10^-17
// 支持范围 Number.MIN_VALUE ~ Number.MAX_VALUE
// 超出范围的将被转换为 -infinity / infinity
    - NaN - 0/0 返回 NaN，非0/0 返回 +-Infinity - 任何涉及 NaN 的操作将返回 NaN，如 NaN/10 - 不等于包括自己在内的任何值 NaN == NaN 返回 false 但是可以通过 isNaN() 进行判断，不能转化为数值的内容均返回 true "10" 可以转为 10，返回 false；"blue"不能转为数值，返回 true - 数值转换函数 - Number() 可用于任何数据类型 - true = 1, fasle = 0 - null = 0 - undifined = NaN - 字符串 - 忽略前缀 0，包含小数点转为浮点，有效十六进制转十六进制 - 空串 = 0 - 包含其他字符 = NaN - 对象 调用 valueOf 转换返回值，结果为 NaN 则调用 toString 按字符串规则转换 - parseInt()主要用于字符串，忽略前缀空格 - 若首个非空字符不是 +- / 数字，则立即返回 NaN（空串也返回 NaN - 从首个非法字符开始忽略后续字符，包括第二个小数点及其后的所有字符 - 支持识别 十进制 / 八进制 / 十六进制 - 支持使用第二个参数指定进制，此时可以省略前缀 - parseFloat()
    - 只能用于十进制，不能指定底数 - 因为会忽略所有前缀 0，十六进制始终返回 0 - 对于整数对象会返回整数结果

    String

    表示 0 或 多个 Unicode 字符，可以使用 双引号 / 单引号 / 反引号 包裹 - 一些转义字符 可以出现在字符串的任意位置，作为单个字符被解释（当成 len = 1 处理） - \b 退格 - \r 回撤 - \f 换页 - \xnn 十六进制编码 nn 对应的 ASCII 字符 - \unnnn 十六进制编码 nnnn 对应的 Unicode 字符 - 字符串不可变，修改字符串 == 销毁原始字符串并创建新值 - 转换为字符串 var.toString() - null / undefined 没有 toString 方法 null 返回 "null", undefined 返回 "undefined" - 对数值对象进行调用的话，可以传入一个参数作为底数 - 模版字面量 - 支持跨行定义字符串（会保持引号内的所有空格） - 保留换行符 - 支持通过 ${} 实现字符串插值，所有插入值将强制使用 toString 转为字符串 - 支持在 ${} 调用自己以插入旧值 - 模版字面量标签函数
