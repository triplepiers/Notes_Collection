    黑马优购 Uni App

    Uni-App 配置与初始化

    Uni-App 是一个使用 Vue 开发前端应用的框架，编写一套代码即可发布到 IOS / Android / 小程序等多个平台，详情见：Uni-App 官方文档

    1.1 下载 & 安装 HBuilderX

    • 下载
      1. 访问 HBuilderX官网
      2. 下载 正式版 -> App开发版
    • 安装 解压后放在纯英文路径下，双击 exe 文件即可运行
    • 安装 scss/sass 编译插件
      1. 访问插件主页
      2. 选择 使用 HBuilderX 导入插件

    1.2 新建 Uni-App 项目

    1. 选择 “文件” - “新建” - “项目”
    2. 填写项目基本信息（项目类型 = uni-app，模版 = uni-ui项目）
    3. 项目基本目录结构如下 
       ┌─components            uni-app组件目录
 │  └─comp-a.vue         可复用的a组件
 ├─pages                 业务页面文件存放的目录
 │  ├─index
 │  │  └─index.vue       index页面
 │  └─list
 │     └─list.vue        list页面
 ├─static                存放应用引用静态资源（如图片、视频等）的目录，注意：静态资源只能存放于此
 ├─main.js               Vue初始化入口文件
 ├─App.vue               应用配置，用来配置小程序的全局样式、生命周期函数等
 ├─manifest.json         配置应用名称、appid、logo、版本等打包信息
 └─pages.json            配置页面路径、页面窗口样式、tabBar、navigationBar 等页面类信息

    1.3 将 uni-app 项目运行到微信开发者工具

    1. 在 manifest.json - 微信小程序配置 中填写小程序 AppID
    2. 在 “工具” - “设置” - “运行配置” - “小程序运行配置”中 设置 “微信开发者工具路径”
    3. 在 微信开发者工具 中，通过 “设置” - “安全设置”，开启“服务端口”
    4. 选择 “运行” - “运行到小程序模拟器“ - ” 微信开发者工具“，编译当前项目并运行

    项目文档

    1 TabBar 及相关页面配置

    1.1 创建页面

    TabBar 中共涉及 home 主页，cate 分类，cart 购物车，my 我的 四个页面。

    1. 在 pages 目录中右键选择“新建页面”
    2. 勾选“scss模版”、“在pages.json中注册“、“创建同名目录”

    1.2 配置 TabBar 样式

    静态资源存放在 ./static 目录下

    修改根目录下的 pages.json，新增 tabBar 配置节点： 

    {
"tabBar": {
    "selectedColor": '#??????',
    "list": [ // list 中的页面顺序需要与 pages 中一致
        {
            "pagePath": "pages/home/home",
            "text": "首页",
            "iconPath": "static/xxx/xxx",
            "selectedIconPath": "static/xxx/xxx"
        }, ...
    ]
}
}

    2 首页实现

    2.1 配置网络请求

    • 原生的 wx.request() API 不支持拦截器等全局配置的功能此处使用 @escook/request-miniprogram 第三方包发起网络请求。安装文档
    • 此外，需要在项目的 main.js 中进行如下配置 
      import {$http} from '@escook/request-miniprogram';
uni.$http = $http;
$http.baseUrl = 'https://www.uniav.com'; // 配置请求根路径
// 请求开始前的相关配置
$http.beforeRequest = function(options) {
    uni.showLoading({ title: '数据加载中...' });
}
// 请求结束后的相关配置
$http.afterRequest = function() {
    uni.hideLoading();
}
