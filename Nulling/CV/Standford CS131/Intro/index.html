1 Linear Algebra

1.1 Vectors & Martices

  • Vector

    • Row Vector: 大小为 1*N

    • Column Vector: 大小为 N*1，转置 (transpose) 后等价于 Row Vector

    在本课程中默认指代 Column Vector

    • 向量内积 Inner Product

      对两个同维 Vector 进行运算，本质是 \(\left|x\right|\left|y\right| \cos\Theta\)

      \[ X^T \cdot Y = \sum_{i=1}^n x_iy_i \]

  • Matrix

    一个 M*N 的矩阵，当 M == N 时称 Square

    在本课程中使用 Matrix 描述图像，以左上角为原点 (0,0)

    • Operations

      1. Addition

        • 对于具有同样大小的矩阵：对应位置相加

        • 对于常数：每个位置加上常数

      2. Scale

        每个位置乘以常数

      3. Product

        只有当矩阵满足：A = M * N & B = N * P 时合法

1.1.1 Vector Norm

向量范数（Vector Norm）是用来衡量向量大小或长度的一种度量方式。它是将向量映射到非负实数的函数。

  • 欧几里德范数（L2范数）：

    \[ \|x\|_2 = \sqrt{\sum_{i=1}^{n} |x_i|^2} \]

其中，\(x_i\)表示向量\(x\)的第\(i\)个元素。

  • 曼哈顿范数（L1范数）：

    \[ \|x\|_1 = \sum_{i=1}^{n} |x_i| \]

曼哈顿范数计算的是向量中各个元素绝对值之和。

  • 无穷范数（L∞范数）：

    \[ \|x\|_{\infty} = \max_{1\leq i \leq n} |x_i| \]

无穷范数计算的是向量中绝对值最大的元素。

向量范数具有以下性质：

  • 非负性：对于任意向量\(x\)，其范数大于等于0，即\(\|x\|\geq 0\)。

  • 齐次性：对于任意向量\(x\)和标量\(\alpha\)，有\(\|\alpha x\| = |\alpha|\|x\|\)。

  • 三角不等性：对于任意向量\(x\)和\(y\)，有\(\|x+y\| \leq \|x\| + \|y\|\)。

  • Definiteness：当且仅当 x = [0,0,0...,0] 时 \(\|x\| = 0\)。
